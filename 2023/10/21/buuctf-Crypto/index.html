[BJDCTF2020]RSA

题目:

分析：

将题目分为2个过程，第一个为加密flag 第二个为加密BJD*32。其中使用了相同的e，而两次加密的n的q相同。则可以通过数学得到两次加密的p，q。
$$
q = gcd(n1,n2)\
p1 = n1 // q\
p2 = n2 // q
$$
再通过第二个加密过程中只未知e，e的范围在（0，100000）之间，爆破e

得到e后，可以通过已知信息解密第一个过程

[BJDCTF2020]rsa_output

[NPUCTF2020]EzRSA

n 可直接分解，但e与phi不互素
$$
c = m^emodn
$$
转化为
$$
c = m^2**e/2 ~~mod~n
$$
后e与phi互素

即可解密

[BJDCTF2020]easyrsa

z=Fraction(1,Derivative(arctan(p),p))-Fraction(1,Derivative(arth(q),q))

得到
$$
z = p ^ 2 - q ^ 2
$$
则可通过

x = gmpy2.iroot(z + 2 * n,2)[0]

y = gmpy2.iroot(z - 2 * n,2)[0]

p = (x + y) // 2

q = n // p

得到q,p。进而得到flag
